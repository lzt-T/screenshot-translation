# screenshot-translation

一个使用 Electron、React 和 TypeScript 构建的 AI 截图翻译工具。

## 功能特性

*   [ ] 快速截图屏幕区域
*   [ ] 图片文字识别 (OCR)
*   [ ] 调用 **AI 引擎**将识别出的文字翻译成目标语言
*   [ ] 在截图区域附近显示翻译结果


## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
